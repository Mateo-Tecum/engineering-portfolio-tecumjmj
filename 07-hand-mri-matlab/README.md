# Hand MRI MATLAB Project

## Overview

This project involved designing and prototyping a small-scale MRI-inspired system intended to image or represent features of a human hand.

The project combined MATLAB analysis, CAD design, system prototyping, and technical presentation to explore how engineering principles can be applied to biomedical imaging systems.

## Key Features

- Biomedical imaging-inspired design
- MATLAB-based analysis or signal processing
- CAD-supported system design
- Physical prototype development
- Technical presentation and documentation
- Interdisciplinary engineering application

## Engineering Skills Used

- MATLAB
- CAD
- Biomedical engineering concepts
- Signal analysis
- System design
- Prototyping
- Technical communication

## Design and Build Process

The project began with the goal of building a working MRI-inspired system focused on the hand. The design process involved researching the concept, developing system components, using MATLAB for analysis or processing, creating CAD models, and assembling a final prototype.

## Results

The final project produced a working prototype and technical presentation. The project strengthened my understanding of how mechanical, electrical, computational, and biomedical concepts can combine in complex engineering systems.
