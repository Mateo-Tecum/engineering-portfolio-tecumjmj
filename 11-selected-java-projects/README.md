# Selected Java Projects

## Overview

This folder contains selected Java programming projects demonstrating object-oriented programming, algorithms, data structures, and software design fundamentals.

The goal is to show programming ability in a clean, organized way without uploading an unfiltered archive of assignments.

## Featured Areas

### Object-Oriented Programming

Projects involving classes, objects, inheritance, encapsulation, and reusable program structure.

### Data Structures

Projects involving arrays, lists, maps, stacks, queues, trees, or other structures used to organize and process data.

### Algorithms

Projects involving searching, sorting, recursion, simulation, or problem-solving logic.
