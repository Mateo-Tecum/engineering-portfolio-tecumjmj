# Welded Truss Structural Analysis

## Overview

This project involved designing, fabricating, and analyzing a welded truss structure using statics and mechanics of materials principles.

The goal was to create a physical truss, calculate expected internal member forces and stresses, and evaluate the structure based on engineering analysis and fabrication quality.

## Key Features

- Physical welded truss fabrication
- Structural analysis using statics
- Stress and force calculations
- Engineering report documentation
- Comparison between design assumptions and physical construction
- Hands-on manufacturing and testing experience

## Engineering Skills Used

- Statics
- Mechanics of materials
- Structural analysis
- Stress calculations
- Welding
- Fabrication
- Technical reporting
- Engineering design documentation

## Design and Build Process

The truss was designed based on structural loading requirements and then fabricated through welding. The analysis included calculating member forces, evaluating stress levels, and documenting whether the structure met expected design behavior. The project connected classroom mechanics concepts with a physical fabricated structure.

## Results

The final truss demonstrated the relationship between design, fabrication, and structural performance. The project strengthened my ability to analyze load paths, calculate stresses, and understand how real manufacturing affects theoretical assumptions.
