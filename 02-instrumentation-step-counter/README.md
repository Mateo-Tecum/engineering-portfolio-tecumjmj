# Instrumentation Step Counter

## Overview

This project used sensor-based instrumentation methods to develop a step counter using accelerometer data and LabVIEW-based signal analysis.

The goal was to collect motion data, process the signal, identify walking patterns, and estimate step count from measured acceleration.

## Key Features

- Accelerometer-based motion measurement
- LabVIEW-based data acquisition and/or signal processing
- Step detection logic
- Sensor calibration and testing
- Experimental data collection
- Comparison between measured motion and expected walking behavior

## Engineering Skills Used

- LabVIEW
- Instrumentation
- Accelerometer data analysis
- Signal processing
- Sensor testing
- Data collection
- Experimental validation
- Technical reporting

## Design and Build Process

The project involved collecting acceleration data during walking motion and analyzing the signal to identify repeated step patterns. The system required interpreting noisy sensor data, selecting appropriate thresholds or detection methods, and testing whether the step counter produced reasonable results across trials.

## Results

The final system was able to estimate step count from accelerometer data using instrumentation and signal analysis methods. The project demonstrated how raw sensor measurements can be converted into useful engineering data through filtering, calibration, and analysis.
