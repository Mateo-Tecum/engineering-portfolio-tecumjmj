# Mechatronic Robot Dog

## Overview

This project is an Arduino-based robotic dog prototype built to demonstrate mechanical design, sensor integration, embedded control, and interactive mechatronic behavior.

The system used a CAD-designed frame, servo actuation, joystick input, ultrasonic distance sensing, and an LCD display to create a small interactive robotic system.

## Key Features

- CAD-designed mechanical frame
- Multiple servo motors for actuation
- Joystick-based user input
- Ultrasonic distance sensing
- LCD display for visual output
- Arduino-based control logic
- Physical prototyping, wiring, debugging, and system integration

## Engineering Skills Used

- Arduino programming
- CAD and mechanical design
- Circuit wiring
- Sensor integration
- Servo motor control
- Embedded systems
- Physical prototyping
- Debugging and testing

## Design and Build Process

The project began with a mechanical frame design, followed by integration of electronic components including servos, sensors, display hardware, and joystick input. The system was assembled, wired, programmed, and tested through iterative debugging.

## Results

The final prototype demonstrated interactive motion and sensor-based behavior using Arduino control. The project required combining mechanical design, electrical wiring, embedded programming, and physical testing into one working system.
