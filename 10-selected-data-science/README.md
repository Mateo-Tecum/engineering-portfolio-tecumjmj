# Selected Data Science Projects

## Overview

This folder contains selected data science and machine learning projects involving exploratory data analysis, modeling, visualization, and evaluation.

The goal is to document technical work that demonstrates programming, data analysis, model development, and communication of results.

## Featured Work

### Exploratory Data Analysis

Projects involving dataset cleaning, visualization, feature exploration, and interpretation of trends.

### Machine Learning Models

Projects involving classification, model training, performance evaluation, and threshold analysis.

### Data Science Internship Work

Selected project work involving data analysis, research, business logic, benchmarking, or applied machine learning.
