# Syringe Pump Prototype

## Overview

This project involved designing, building, and programming a syringe pump prototype using CAD, Arduino-based control, electronics, and physical fabrication.

The goal was to create a controlled actuation system capable of moving a syringe plunger in a repeatable way.

## Key Features

- CAD-designed mechanical structure
- Arduino-based control logic
- Motor or actuator-driven syringe motion
- Circuit wiring and soldering
- Physical prototype assembly
- Testing and debugging of mechanical and electrical systems

## Engineering Skills Used

- CAD
- Arduino programming
- Circuit wiring
- Soldering
- Mechanical prototyping
- Actuator control
- 3D printing or fabrication
- System testing

## Design and Build Process

The project began with the design of a mechanical structure to hold and actuate a syringe. The prototype required integrating electronics, control logic, and physical components so that the syringe plunger could move in a controlled manner. The build process involved CAD design, fabrication, wiring, programming, and iterative debugging.

## Results

The final prototype demonstrated controlled syringe actuation through an integrated mechanical and electronic system. The project introduced me to the challenges of combining physical hardware, electronics, and code into a working prototype.
