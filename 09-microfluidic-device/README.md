# Microfluidic Device Prototype

## Overview

This project involved designing a microfluidic device using CAD and preparing the design for resin 3D printing.

The goal was to create a small-scale fluidic device concept and explore how CAD and additive manufacturing can be used for prototyping biomedical or laboratory systems.

## Key Features

- CAD-designed microfluidic geometry
- Fusion 360 design workflow
- Resin 3D printing preparation
- Small-scale prototyping
- Biomedical device concept
- Design-for-manufacturing considerations

## Engineering Skills Used

- Fusion 360
- CAD design
- Additive manufacturing
- Resin 3D printing
- Microfluidics concepts
- Biomedical prototyping
- Design iteration

## Design and Build Process

The project began with designing a microfluidic geometry in CAD. The design was prepared for resin printing and reviewed for manufacturability, feature size, and expected fluid pathway behavior. The project emphasized precision, small-scale design, and practical limitations of 3D printing.

## Results

The final design was resin printed as a microfluidic device prototype. The project helped me understand how CAD can be applied to biomedical device prototyping and how manufacturing constraints affect small-scale fluidic designs.
