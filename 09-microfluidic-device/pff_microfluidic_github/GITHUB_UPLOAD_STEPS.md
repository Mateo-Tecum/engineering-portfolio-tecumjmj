# Upload steps for GitHub

1. Create a new GitHub repository named something like `pff-microfluidic-design`.
2. Upload the contents of this folder, not the ZIP file itself.
3. Keep `README.md` at the top level so GitHub displays it automatically.
4. Add CAD screenshots or exported STL/STEP files later under an `cad/` or `assets/` folder if you still have them.
5. Do not upload full journal PDFs unless you have permission to redistribute them. Keep the citations in `references/references.bib` instead.

Suggested repository description:

> Concept design for a 3D-printable pinched-flow-fractionation microfluidic device for size-based polymer bead separation.
